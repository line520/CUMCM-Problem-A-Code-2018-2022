本文档介绍这篇论文用到的所有代码
======= 数据拟合和计算 ======
simulate.m              拟合各个变量之间的函数关系以及绘制凸轮喷油速率等图像
calP.m                  利用密度计算压强的函数(多项式拟合求解)
calrho.m                利用压强计算密度的函数(多项式拟合求解)
calrho_iter.m           利用压强计算密度的函数(利用迭代求解)
Height.m                绘制凸轮极径和柱塞上升高度对比图
injection.m             绘制喷油嘴喷油面积变化图
======== 问题一 ==========
question1.m             计算得到最优的单向阀打开时间和喷油开始时刻
======== 问题二 ==========
question2.m             计算得到最优的凸轮旋转角速度
======== 问题三 ==========
question3_injection.m   给出最优的凸轮旋转角速度和两个喷油嘴的控制策略
question3_tube.m        给出在两个喷油嘴和一个减压阀下凸轮最优旋转角度和控制策略
