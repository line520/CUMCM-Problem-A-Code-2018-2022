本文档介绍各个代码的功能
==========  第一问  =========
getparameter.m     用来获取传送带上任意位置的环境温度、换热系数、导热系数
heatmodel.m        获取指定参数和步长下的电子元件的温度分布
thomas.m           利用追赶法求解三对角方程组
simulate.m         根据实验数据拟合最佳的换热系数和导热系数
question1.m        求解指定温度和传送带速度下的炉温曲线
==========  第二问  ==========
question2.m        根据实验条件求解最佳的传动带速度
==========  第三问  ==========
question3.m        求解面积最小时对应的各温区设定温度和传送带速度
==========  第四问  ==========
question4.m        求解在尽量要求对称下最优的各温区设定温度和传送带速度
